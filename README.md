# agentic_ai_use_cases

Collection of practical agentic AI use cases built with modern Python tooling. Each project is self-contained with its own README for setup and usage.

## Projects

- `multi_agent_report_generator`: Multi-agent pipeline that researches a topic, analyzes trends, writes and proofreads a report, and saves the final output as a PDF at `multi_agent_report_generator/outputs/final_report.pdf`. See its README for setup and usage.
